
# CS226 Big Data Management Project
This project analyzes crime patterns in Los Angeles using distributed data processing with PySpark and spatial visualization.
It also creates an WEBUI for the Spatial and Analytic Queries using Streamlit

# Data Sets Used
**LA Crime Data** 
https://catalog.data.gov/dataset/crime-data-from-2020-to-present

**Median Income Data**
https://geohub.lacity.org/datasets/lacounty::median-income-and-ami-census-tract/about

# Dev-container: 
I have made this standalone dev-container for this project, which has Pyspark, PostgreSQL in it
